# Visual Style Guide — Barnbocker / Fifi the Fox Series
*Version 1.0 — Creative Director, April 2026*

## Core Aesthetic

**Style name:** Soft Storybook  
**Feel:** Warm, cozy, hand-painted children's book — think Beatrix Potter meets modern Scandinavian illustration. Gentle, not cartoony. Soft, not flat.

**Art technique (AI generation):** Soft watercolor-inspired digital illustration. Slightly textured, as if painted on paper. Avoid hard outlines or harsh shadows.

### Accent Palette (used sparingly)

| Name | Hex | Usage |
|------|-----|-------|
| Berry Red | `#E74C3C` | Berries, apples, rare bold accents |
| Lavender | `#C39BD3` | Nighttime scenes, magic moments |
| Deep Navy | `#2C3E50` | Eyes, noses, fine details only |

### Rules
- Never use pure black (`#000000`) — use Deep Navy for outlines/details
- Background skies should always trend warm or pastel, never cold grey
- At least 60% of every spread should be the Primary Palette

## Illustration Composition

### Page Layout (for 6-page books, iPhone/iPad portrait)
- **Full-bleed illustrations** — image fills the page edge to edge
- Text overlaid at bottom with a soft semi-transparent white band (no hard box)
- **Safe zone for text:** bottom 22% of image height
- Characters placed center-screen or slightly left — leave right side for text overlay in dual-page layouts

### Focal point rule
- One clear subject per page (Fifi, a friend, or a key object)
- Background elements should be soft/blurred to keep eyes on the character
- Avoid busy, cluttered scenes — max 3 distinct elements per page

### Perspective
- **Eye-level or slightly above** — we see the world from a child's height
- No dramatic bird's-eye or worm's-eye unless it's a special "wonder" moment

---

## Character Proportions

All characters use **chibi-adjacent proportions** for child-appeal:
- Head = ~40% of total body height
- Eyes = large, taking up ~30% of the face
- Limbs = short and rounded, no sharp angles
- Hands/paws = mitten-like, not detailed fingers

This applies to Fifi, Bruno, Bella, and Oliver consistently across all books.

## Backgrounds & Environments

### The Forest (primary setting)
- Soft deciduous forest with rounded, friendly trees
- Dappled sunlight effect — warm golden patches on the ground
- Green grass with small wildflowers (white, yellow, pink)
- Avoid: dark shadows, spooky angles, dense undergrowth

### Other approved environments
| Setting | Mood | Key elements |
|---------|------|-------------|
| Meadow | Bright, open | Soft grass, flowers, big sky |
| Riverbank | Calm, curious | Gentle water, smooth pebbles, reeds |
| Cave/den | Cozy, safe | Warm amber light, moss, soft bedding |
| Snow scene | Magical, quiet | Soft blue-white, warm glow from windows |
| Beach | Playful, surprising | Pale sand, calm waves, shells |

### Seasonal palette shifts
- **Spring:** Pastel greens, pink blossoms, soft morning light
- **Summer:** Rich greens, golden sunlight, bright flowers
- **Autumn:** Amber, orange, rust, falling leaves
- **Winter:** Muted blues, warm amber interiors, snow whites

## Typography (for text overlay)

- **Font family:** Rounded sans-serif (e.g., Nunito, Rounded Mplus, or similar)
- **Font size:** Minimum 22px on mobile for ages 1–5 (large print standard)
- **Color:** Deep Navy `#2C3E50` on cream background band
- **Line spacing:** 1.6× minimum — children learning to read need generous spacing
- **Max words per text block:** 20 (aim for 10–15)

---

## What to Avoid

| Avoid | Because |
|-------|---------|
| Sharp geometric shapes | Feel cold, not cozy |
| Realistic animal anatomy | Can feel scary to small children |
| Dark shadows or dramatic lighting | Can be unsettling for 1–3 year olds |
| Complex backgrounds with many elements | Overstimulating, distracts from story |
| Pure black anywhere | Too harsh for the warm palette |
| Text in all-caps | Harder to read, feels aggressive |
| Facial expressions outside approved list | Risk of emotional confusion or fear |

## Prompt Template for Image Generation (Minimax / similar)

```
[SCENE DESCRIPTION]. Soft children's book watercolor illustration style. 
Warm color palette — oranges, creams, soft greens and blues. 
Large expressive dark eyes, rounded friendly shapes, gentle textures. 
Child-friendly and cozy. No sharp edges or dark shadows. 
Bright, warm lighting. Suitable for ages 1–5.
```

Always prepend the character base description from `characters/character-bible.md` before the scene description.
