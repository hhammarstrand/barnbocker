# Character Bible - Fifi the Fox

## Main Character: Fifi the Fox

### Character Overview

**Name:** Fifi the Fox  
**Species:** Red Fox (Vulpes vulpes)  
**Age:** Young (child equivalent, approximately 4-5 years old in personality)  
**Personality:** Curious, friendly, adventurous, kind-hearted, playful  
**Catchphrase:** "What shall we discover today?"

### Physical Description

**Appearance:**
- Bright orange fur with cream-colored belly and snout
- Large, expressive dark eyes
- Fluffy white-tipped tail
- Small to medium size (child-appropriate proportions)
- Occasionally wears a colorful scarf or bow

**Distinguishing Features:**
- Big curious eyes that sparkle with wonder
- Small black nose
- Pointed ears with light inner color
- Soft, rounded features (not sharp or intimidating)

**Color Palette:**
- Primary: #E67E22 (bright orange)
- Secondary: #FDEBD0 (cream)
- Accent: #FFFFFF (white tail tip, belly)
- Details: #2C3E50 (eyes, nose)

### Personality Traits

1. **Curious** - Fifi loves exploring and asking questions
2. **Friendly** - Always ready to make new friends
3. **Brave** - Not afraid of new experiences (but age-appropriate bravery)
4. **Kind** - Shares with friends and helps others
5. **Playful** - Loves games, laughter, and fun

### Voice & Speech

- Speaks in short, simple sentences (appropriate for ages 1-5)
- Uses friendly, warm tone
- Often uses rhyming or repetitive phrases
- Asks lots of questions
- Voice description: "bright, cheerful, childlike wonder"

### Character Arc Themes

- Learning about friendship
- Overcoming small challenges
- Exploring nature and the world
- Understanding emotions
- Sharing and kindness

## Visual Consistency Rules

1. **Always use the base character prompt** as the starting point
2. **Maintain orange/cream color scheme** for Fifi
3. **Keep expressions child-friendly** - happy, curious, surprised (never scary)
4. **Use soft, rounded shapes** - no sharp angles
5. **Warm color palette** - oranges, yellows, soft browns, creams
6. **Consistent eye style** - large, dark, expressive
7. **Same illustration style** across all books (Minimax image-01 model)

---

## Usage in Generation Pipeline

When generating illustrations for a Fifi story:

1. Use the base prompt + scene-specific variation
2. Include "Children's book illustration style"
3. Add character consistency elements from the base prompt
4. Specify warm lighting and soft colors
5. Ensure child-friendly expressions and poses
